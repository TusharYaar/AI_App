#Mobile App for AI project

###pre-requirements
1. Node JS 
2. expo cli
3. Mobile Device


###How to run 
1. Clone the repository
2. run `npm run android` to run on an android device or `npm run ios` to run on an iOS device. 


###Caviats
On emulators (both android and iOS) camera cannot be used, and thus only on device photos can be used.
